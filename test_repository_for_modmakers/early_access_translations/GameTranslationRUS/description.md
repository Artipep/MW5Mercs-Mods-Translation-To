# Game Translation RUS v1.13

Translation of the base game into Russian. Compatible with game version 1.13.378.

## Description
This is a test translation mod that targets the original game rather than a specific mod.
When `for_mod_name` is set to `"Original_game"`, the program checks compatibility
against the game version configured in program settings.

## Compatibility
- MechWarrior 5: Mercenaries v1.13.378
